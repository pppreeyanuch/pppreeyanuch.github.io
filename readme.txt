== Description ==
Personal Club is a simple, clean, easy to use and user-friendly free personal blog WordPress theme. This theme is developed for to create personal or blogging websites. It can also be used for fashion blog, portfolio, beauty & spa, lifestyle, news, travel, photography and food blog websites. This theme is compatible with many popular WordPress plugins like Contact Form 7, Gravity forms, WooCommerce, NextGEN gallery, Revolution slider, Cyclone slider, Jetpack, Yoast
